# data/

Place `Data PMI.xlsx` and `All Droping.xlsx` in this folder before running
`run_historical.py` or `run_live.py`.

These files are **not committed to git** (see `.gitignore`) because they contain
real donor records, including blood-screening results (HIV/HCV/HBsAg/Syphilis
status). Handle them the same way you'd handle any other patient health data:
don't email them unencrypted, don't upload them to a public repo or shared
drive, and don't paste rows into chat tools or tickets.
